# Metryx — Dashboard (static)

Self-contained rebuild of the Metryx performance dashboard. Pure static
HTML/CSS/JS, **zero build step and zero runtime dependencies** (Google Fonts
loaded with a system-font fallback, so it also works offline).

> This is a standalone static artifact. It is **not** the Next.js source of the
> production app (`metryx-app`). It was rebuilt from a saved snapshot of the
> dashboard page into a single, fully-interactive static site.

## Features

- KPI cards (Investimento / Receita / ROAS / CPL) with period-over-period deltas and sparklines
- Receita × Investimento combo chart (SVG, animated, hover tooltip, legend toggle)
- Conversion funnel (Impressões → Cliques → Leads)
- Investment-by-channel breakdown
- Campaigns table sorted by ROAS
- AI insights drawer (derived from the current dataset)
- Range switch (7 / 30 / 90 days), client selector, metrics picker (max 4)
- Dark / light theme (persisted), shareable deep-links (`?client=&range=`)
- Responsive: desktop, tablet, mobile

Data is deterministic mock data generated by a seeded PRNG keyed by
`(client, range)`, so every widget stays mutually consistent.

## Run locally

Open `index.html` directly, or serve the folder:

```bash
python -m http.server 5500
# http://localhost:5500
```

## Deploy

Pure static — deploy the folder as-is to Vercel, Netlify, or GitHub Pages.
No framework preset, no build command, output directory = repo root.
